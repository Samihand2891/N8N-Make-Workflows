A collection of automation workflows built with **n8n** and **Make.com**.

## 📋 Overview

This repository contains reusable automation workflows for n8n (self-hosted/low-code automation tool) and Make.com (formerly Integromat).

These workflows help automate various tasks across different tools and services.
